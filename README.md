## 🎮 Sobre o Jogo
O **Neon Puzzle Pro** desafia o jogador a navegar por labirintos de neon, onde o objetivo é alcançar o ponto de destino com o menor número de movimentos possível. O jogo apresenta uma curva de aprendizado suave, introduzindo novos desafios a cada nível concluído.

### 🚀 Funcionalidades
* **Progressão Dinâmica:** Fases que aumentam de complexidade conforme você avança.
* **Visual Neon:** Interface construída com efeitos de brilho (Glow Effects) via Canvas API.
* **Sistema de Colisão:** Lógica robusta de detecção de paredes e limites de mapa.
* **Design Modular:** Código separado em HTML, CSS e JS para fácil manutenção.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura e elemento `<canvas>`.
* **CSS3:** Estilização moderna, variáveis e animações de interface.
* **JavaScript (ES6+):** Motor lógico, gerenciamento de estados e renderização.
